1. Selectores jerárquicos

Para diversificar las opciones, CSS3 ofrece multitud de selectores.

Los selectores jerárquicos son:

A B Descendentes Selecciona cualquier elemento B que es descendiente del elemento A
A > B Hijos Selecciona cualquier elemento B que es hijo directo del elemento A
A + B Hermano adyacente Selecciona cualquier elemento B precedido inmediatamente del elemento A

2. Pseudoclases de selección

Las pseudoclases de selección son:

:root - Representa el elemento raíz o el elemento en el nivel más alto del documento. Por ejemplo, en HTML5, la etiqueta <html>.
:empty - corresponde a los elementos vacíos y que no tienen hijos.
:only-child - devuelve el hijo único, es decir, este elemento no tiene elementos hermanos.
:first-child - selecciona el primer elemento hijo.
:last-child - selecciona el último elemento hijo.
:nth-child - designa el enésimo elemento hijo, donde n es un número o bien las palabras clave even (pares) y odd (impares).
:nth-last-child(n) - selecciona el enésimo elemento hijo, a partir del último elemento, donde n es un número o bien las palabras clave even (pares) y odd (impares).
:only-of-type - devuelve el único elemento de este tipo.
:first-of-type - devuelve el primer elemento de este tipo.
:last-of-type - devuelve el último elemento de este tipo.
:nth-of-type(n) - designa el enésimo elemento de este tipo, donde n es un número o bien las palabras clave even (pares) y odd (impares).
:nth-last-of-type(n) - selecciona el enésimo elemento de este tipo partiendo del último elemento, donde n es un número o bien las palabras clave even (pares) y odd (impares).

3. Selectores de atributos

Los selectores de atributos son:

[atr] - Designa un elemento que contiene el atributo atr informado.
[atr="valor"] - Designa un elemento que contiene el atributo atr con el valor indicado.
[atr~="valor"] - Corresponde con todo elemento donde el atributo atr contenga una lista de valores separados por espacios y donde alguno de ellos sea dicho valor.
[atr|="valor"] - Corresponde con todo elemento donde el atributo atr contenga una lista de valores separados por guiones comenzando por el valor.
[atr^="valor"] - Representa un elemento donde el atributo atr comience con el prefijo indicado.
[atr$="valor"] - Representa un elemento donde el atributo atr termine con el sufijo indicado.
[atr*="valor"] - Representa un elemento donde el atributo atr contenga una instancia u ocurrencia del valor indicado.
